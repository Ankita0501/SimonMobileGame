Simon Game Rpes

Objective of the Game

Correctly repeat a longer and longer sequence of signals.

1-player Game

Press the START button.

Simon will give the first signal.

Repeat the signal by pressing the same color lens.

Simon will duppcate the first signal and add one.

Repeat these two signals by pressing the same color buttons,in order.

Simon will duppcate these first two signals and add one.

Continue playing as long as you can repeat each sequence of signals correctly.

If you fail to repeat a sequence exactly,Simon responds with a "RAZZ" sound.

This means you've lost, and the sequence of signals ends.
